# Active-Directory-Home-Lab

This project documents the setup of a fully functional Active Directory environment running on a virtualized Windows Server 2022 instance. The lab simulates a real-world enterprise environment and was built to develop and demonstrate skills relevant to IT support, systems administration, and cybersecurity roles.


 Technologies Used

Hypervisor: VirtualBox / VMware (update with what you used)
Server OS: Windows Server 2022
Directory Service: Active Directory Domain Services (AD DS)
Management Tools: Active Directory Users and Computers (ADUC), Server Manager



🏗️ What Was Built

Virtual Machine Setup

Installed and configured Windows Server 2022 as a virtual machine
Allocated resources (RAM, CPU, storage) and configured network settings
Promoted the server to a Domain Controller

Active Directory Domain Services (AD DS)

Installed and configured the AD DS role
Created and managed a local Active Directory domain
Configured DNS integration required for AD functionality

 Organizational Units (OUs)

Designed an OU structure to mirror a real enterprise layout
Created OUs to logically separate and manage different departments or groups (e.g., IT, HR, Management)
Applied delegation of control concepts within OUs

Users & Groups

Created multiple user accounts with realistic attributes (name, logon name, department)
Organized users into appropriate Security Groups and Distribution Groups
Applied the principle of least privilege when assigning group memberships
